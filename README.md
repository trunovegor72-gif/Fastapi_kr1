## Установка и запуск

```bash
pip install -r requirements.txt
uvicorn app:app --reload
```

После запуска:

- Корневая HTML-страница: <http://localhost:8000/>
- Интерактивная документация Swagger: <http://localhost:8000/docs>

Проверить все маршруты автоматически:

```bash
python test_endpoints.py
```

## Описание выполнения заданий

### Задание 1.1 — корневой JSON-ответ

Создаётся FastAPI-приложение, которое возвращает JSON-приветствие.

Поскольку в **задании 1.2** маршрут `/` переопределяется на отдачу HTML,
JSON-приветствие из 1.1 вынесено на отдельный явный маршрут `/welcome`:

```
GET /welcome  ->  {"message": "Добро пожаловать в моё приложение FastAPI!"}
```

**Дополнительные пункты задания 1.1:**

1. Запуск в режиме разработки с автоперезагрузкой:

   ```bash
   uvicorn app:app --reload
   ```

2. *Проверка авторелоада.* Без остановки `uvicorn` нужно изменить текст ответа
   на `{"message": "Авторелоад действительно работает"}` и убедиться через
   браузер, что новое сообщение возвращается без перезапуска сервера.
   Флаг `--reload` отслеживает изменения в файлах и автоматически
   перезагружает приложение.

3. *Переименование переменной приложения.* Если переименовать переменную,
   например `app` → `my_app`, то uvicorn потребует другой аргумент запуска:

   ```bash
   uvicorn app:my_app --reload
   ```

   То есть формат запуска — `uvicorn <имя_файла>:<имя_переменной>`.

```
POST /user
тело: {"name": "Egor Trunov", "age": 26}

```

Поле `is_adult` равно `true`, если возраст ≥ 18, иначе `false`.

### Задания 2.1 и 2.2* — отзывы с валидацией

Маршрут `POST /feedback` использует модель `FeedbackValidated`, которая
объединяет требования простого варианта (2.1) и расширенного (2.2*):

- `name` — обязательное, от 2 до 50 символов;
- `message` — обязательное, от 10 до 500 символов;
- кастомный валидатор `@field_validator` отклоняет сообщения с
  недопустимыми словами **«кринж», «рофл», «вайб»** в любых падежах
  (поиск по корню слова через регулярное выражение, без учёта регистра)
  и выбрасывает `ValueError("Использование недопустимых слов")`.

При успехе:

```
POST /feedback
тело: {"name": "Артур", "message": "Это тяжело, но я справлюсь!"}
ответ: {"message": "Спасибо, Артур! Ваш отзыв сохранён."}
```

Отзыв сохраняется в список `feedbacks`. Посмотреть все сохранённые отзывы
можно через вспомогательный маршрут `GET /feedbacks`.

При нарушении ограничений FastAPI/Pydantic автоматически возвращает
**HTTP 422** с подробным описанием ошибок (как в примере из задания).
